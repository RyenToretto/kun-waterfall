## ADDED Requirements

### Requirement: 瀑布流布局

系统 SHALL 以瀑布流布局展示视频卡片，支持随机尺寸形成错落效果。

#### Scenario: 瀑布流多列布局

- **WHEN** 视频数据加载完成
- **THEN** 视频卡片按瀑布流布局排列
- **AND** 使用 CSS Grid 实现多列布局
- **AND** 列数根据响应式规则动态调整

#### Scenario: 卡片随机尺寸

- **WHEN** 视频卡片渲染时
- **THEN** 组件应随机生成卡片高度比例
- **AND** 大部分卡片尺寸相同，部分卡片随机变化
- **AND** 形成视觉上的瀑布流错落效果

#### Scenario: 视频内容裁剪

- **WHEN** 视频内容铺满卡片
- **THEN** 竖屏视频（高度大于宽度）截取顶部部分
- **AND** 横屏视频（宽度大于高度）截取中间部分
- **AND** 内容始终铺满卡片区域，无黑边

### Requirement: 视频卡片组件

系统 SHALL 提供视频卡片组件，展示视频预览和基本信息。

#### Scenario: 视频预览显示

- **WHEN** 卡片进入可视区域
- **THEN** 显示视频内容（封面或播放中的视频）
- **AND** 视频内容根据裁剪规则适配卡片尺寸

#### Scenario: 卡片交互扩展

- **WHEN** 用户与卡片交互
- **THEN** 卡片右上角可显示收藏按钮（可扩展需求）
- **AND** 预留交互扩展接口

### Requirement: 无限滚动加载

系统 SHALL 支持滚动到底部时自动加载更多视频。

#### Scenario: 触底加载更多

- **WHEN** 用户滚动到页面底部附近
- **THEN** 自动触发加载下一页数据
- **AND** 新数据追加到瀑布流末尾
- **AND** 保持滚动位置不跳动

#### Scenario: 加载状态显示

- **WHEN** 正在加载更多数据
- **THEN** 显示加载中状态
- **AND** 加载完成后自动隐藏

#### Scenario: 数据加载完毕

- **WHEN** 所有数据已加载完毕
- **THEN** 不再触发加载请求
- **AND** 可显示"已加载全部"提示
